# Umihara Kawase - No Random Enemy Spawns
A mod for the PC release of Umihara Kawase that disables random enemy spawns. The mod doesn't remove enemies completely, the ones that were manually placed by the developers will still appear in the game.

## Installation
- Download **dinput8.dll** from the [Releases](https://github.com/digital-threat/UmiharaKawaseNoRandomSpawns/releases/latest) page
- Place **dinput8.dll** in the same folder as the game executable (UmiFstWin.exe)

## Uninstallation
- Delete **dinput8.dll** from the game folder
